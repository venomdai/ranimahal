# Block & Tile Works Website

A simple static website for a concrete block and tile business.

## Files
- `index.html` — main landing page
- `styles.css` — responsive styling
- `script.js` — mobile menu and contact form behavior

## Usage
1. Open `index.html` in a browser.
2. Edit the text, colors, and product information to match your business.
3. Replace placeholder sections with your logo and real images as needed.
